What is Big-O?

Big-O

1. O(1): Constant access time
2. O(log N): Logarithmic access time
3. O(N): Linear access time
4. O(N^2): Quadratic access time
5. O(2^N): Exponential access time

Big-O Comparison of Complexity
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ) < O(n!) < O(nⁿ)
